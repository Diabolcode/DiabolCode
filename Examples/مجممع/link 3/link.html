What is the best educational plan for you?


First, you should know the types of programmers


1)Front-end development:

What is the job of a front-end programmer?

It is ( client side development ), meaning that everything visitors see on the site is responsible for it,
    He studies the following programming languages:

  1. HTML
  2. CSS
  3. JS
  4. SVG & Canvas
  5. Sass
  6. Raect.js
  7. Git

2)back-end development

What is the job of a back-end programmer?

It is ( server side development ) that is, it is responsible for all commands related to the server such as modifying and saving data,
     protecting the site and user data, modifying the page or creating a new page,
        He studies the following programming languages:

  1. PHP
  2. MySQL
  3. jquery
  4. Python or Java
  5. API
  6. node.js
  7. Git

3)full stack development

What is the job of a full stack web development programmer?

He is a programmer who works as programmers ( front-end and back-end ) and studies what has been studied.
     He entered the two departments,but he is not preferred in the labor market for large companies and often finds
         his opportunity in small companies because large companies are looking for skill and quality and they think that
             you specialize in one field in which you are more skilled, unlike small companies that are looking to reduce the
                 number of employees.Opening the door to hiring one programmer to work as programmers


Which type of web programmer is best if you are a beginner ?

It is the front-end because it is easy and fun , and not as complicated as the back-end .
